# 13 - 06

A personal space by malik yakub, built with Next.js.

## About

This is a minimalist personal website featuring a clean, modern design with a teal accent color scheme. The project uses Next.js 13+ with the App Router for optimal performance and developer experience.

## Credits

UI Design inspired by [Yanka Darelova](https://www.darelova.com/)'s minimalist and elegant design approach.

## Tech Stack

- [Next.js](https://nextjs.org) - React framework
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [Geist Font](https://vercel.com/font) - Typography

## Development

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

- `src/app/` - Main application code
  - `page.tsx` - Home page component
  - `layout.tsx` - Root layout with metadata
  - `globals.css` - Global styles and color variables
